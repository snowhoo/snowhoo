## 01 补足镁能让你睡得更好

{% gallery %}
![](https://snowhoo.net/images/posts/20260513_230451/1.png)
{% endgallery %}

这项发表在《美国临床营养学杂志》上的研究，由我国复旦大学和美国马萨诸塞大学洛厄尔分校、塔夫茨大学团队共同完成。

{% gallery %}
![](https://snowhoo.net/images/posts/20260513_230451/2.png)
{% endgallery %}

研究人员通过问卷评估了613名参试者的膳食镁摄入量，并通过血检测定血清镁水平，随访时间为6年。

结果发现，**在无糖尿病的健康人群中，每日额外多摄入100毫克镁，失眠风险可降低58%**，补镁对缓解入睡困难、早醒等常见睡眠问题效果最为突出。

研究还指出，无论是通过天然食物还是膳食补充剂补镁，对睡眠的改善作用都差不多，这也提示**普通人完全可以通过日常饮食，安全且温和地提升睡眠质量**。

不过研究同时提醒，这一结论并不适用于所有人。**对于糖尿病患者而言，盲目补镁不仅难以改善睡眠，反而可能产生不利影响。**

{% gallery %}
![](https://snowhoo.net/images/posts/20260513_230451/3.png)
{% endgallery %}

> 对于其中的机制，有学者认为，镁缺乏会导致大脑中一种名为**伽马氨基丁酸（GABA）**的神经递质减少，这种物质与放松、睡眠密切相关。
