# Hexo 邮件发布模板

> 发送至：**9187541@qq.com**
> 主题必须以 `@hexo` 开头才会被识别发布

---

## ✅ 通用发布模板

```
收件人：9187541@qq.com
主题：@hexo 文章标题

这是文章正文内容，支持多段落。

#标签1 #标签2 #标签3

[cat:分类名]
```

---

## 📷 带图片模板（第一张自动设为封面）

```
收件人：9187541@qq.com
主题：@hexo 我的旅行日记

今天去了海边，天气特别好，海风很舒服。

#旅行 #生活 #记录

[cat:随笔]
```

> 💡 附件直接添加图片即可，无需在正文写图片路径
> 第一张图片自动设为文章封面（列表页显示）
> 其余图片自动插入文章正文

---

## 📋 规则速查

| 功能 | 格式 | 示例 |
|------|------|------|
| 发布标识 | `@hexo` 开头 | `@hexo 我的文章` |
| 标题 | 自动从主题提取 | 主题文字即标题 |
| 正文 | 任意文字 | 多行不限 |
| 标签 | `#标签名` | `#Hexo #自动化` |
| 分类 | `[cat:分类名]` | `[cat:技术,生活]` |
| 草稿 | 正文含 `[草稿]` | 文章存入草稿箱 |
| 图片 | 邮件直接附图 | 自动处理，封面+正文 |
